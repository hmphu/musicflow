## 🎵 Featured Project — MusicFlow

> **Spotify-style Chrome extension that plays YouTube music with no ads, no login, and zero API quota limits.**

- 🔍 Search powered by **yt-dlp** — unlimited, no API key needed
- 🎵 **Audio-only** playback — no YouTube tab ever opens
- 💾 **Playlists** — save queues, import any YouTube URL
- 🎤 **Lyrics** — one click
- ⌨️ **Hotkeys** + **Windows media overlay** support
- 🚀 **Auto-starts** on Windows login
